macOS 系统使用技巧之引言

序言

macOS 是苹果电脑的操作系统，我有幸在2015年入手一台Mac Book Pro。在和它朝夕相伴的日子里，我深深的被他强大的功能与精美的设计所折服了。

为什么要写这个系列

对于新系统的探索我从来没有停止过，我查过很多资料，同样的我也遇过很多坑。但是这些内容都没有被仔仔细细的整理过，也没有形成一个系统的知识体系。我想把我所知道的关系苹果系统的各种小技巧整理出来。成系统的给大家看。

关于引用

我不是苹果，我所有的知识都是来源于网络或者是苹果的官方文档，我在此对于那些文档的作者表示感谢。但是在过去的日子里，我看过的文档不计其数，我不能对每一遍文章的作者标识感谢，也不能标明每一篇的出处。在此我对我以前看过的所有文章的作者标识感谢。我可以保证我的文章都是我一个字一个字打出来的。不会直接拷贝粘贴，这也算是原创了吧。

我的文章写出来就是希望更多的人看到，大家可以引用。但是我希望大家在大篇幅引用的时候可以添加出处。子轩这里谢谢各位。
